# QR-CODE-GENERATOR
It is used to convert the alphanumeric code to the quick response(QR) code.
I used the Google Chart API to convert the alphanumeric code to QR code.
Google chart API consists of three parameters they are 
   1.chart type (cht)
       cht=qr
   2.chart size (chs)
       chs=150x150
   3.chart data (chl)
      chl=some data
